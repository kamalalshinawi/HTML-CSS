# Basic HTML & CSS Project

This is a simple project built using **HTML** and **CSS**. It is intended for learning and practicing the fundamentals of web development.

## Project Structure

```
project-folder/
│
├── index.html      # Main HTML file
├── style.css       # Stylesheet
└── README.md       # Project documentation
```

## Features

- Basic HTML page structure
- External CSS styling
- Clean and simple layout
- Beginner-friendly code

## Technologies Used

- HTML5
- CSS3

## Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your web browser.

## Customization

You can modify:
- `index.html` to change the content.
- `style.css` to change the design and layout.

## Learning Objectives

This project helps you practice:

- HTML document structure
- Headings, paragraphs, and links
- Images and lists
- CSS selectors
- Colors and fonts
- Margin and padding
- Basic page styling
